# Локальный запуск (API + демо веб-морда)

Это репозиторий с API для расчёта астрологических данных (положения планет, дома, углы).
API можно вызывать из других программ, например из horoscope-бота или любого другого внутреннего сервиса.

Для наглядной проверки в проекте есть тестовая веб-морда: она удобна для ручного ввода параметров и просмотра результата.

## 1) Куда вставить токен GPT

1. В корне проекта скопируйте шаблон:
```bash
cp secrets.txt.example secrets.txt
```
2. Откройте файл `secrets.txt` и заполните:
```text
OPENAI_API_KEY=sk-...
```

Важно: файл `secrets.txt` добавлен в `.gitignore` и не должен коммититься.

## 2) Как запустить API-сервис

Зайдите в папку проекта и выполните:

```bash
./scripts/start_api_local.sh
```

API будет доступен на `http://127.0.0.1:8013`.

## 3) Как отдельно запустить демо веб-морду

В новом терминале:

Зайдите в папку проекта и выполните:

```bash
./scripts/start_web_ui.sh
```

Веб-морда будет доступна на `http://127.0.0.1:8014`.

## 4) Как открыть

Откройте в браузере:

```text
http://127.0.0.1:8014
```

Нажмите «Сделать»:
- верхнее поле: гороскоп, сгенерированный через OpenAI API;
- нижний raw JSON API скрыт по умолчанию, показывается/скрывается отдельной кнопкой.
