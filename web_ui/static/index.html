Ректификация рождения — мастер-сценарий
Главный путь для пользователя и астролога. Введите данные один раз, затем пройдите шаги по порядку.
Шаг 1 из 5 — Дата и место
Введите дату рождения и город. Система автоматически подставит координаты, timezone и общие настройки.
Шаг 2 из 5 — Asc-интервалы
Мы рассчитаем все возможные восходящие знаки и сохраним их как базу для дальнейшей ректификации.
Шаг 3 из 5 — Диалог по Asc
Ответьте на вопросы о внешности, подаче и стиле. Это помогает выбрать 1–3 наиболее вероятных Asc-кандидата.
Шаг 4 из 5 — События жизни
Добавьте 3–7 важных событий. Они нужны, чтобы сузить время рождения и проверить гипотезы по событиям.
