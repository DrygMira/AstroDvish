# Totem Degree Data (placeholder)

This directory is a placeholder for an optional 360-degree totem database.

## Current status

- Runtime returns only technical `asc_degree_index` and warning `totem_database_not_connected`.
- No copyrighted symbolic descriptions are bundled.

## Integration requirements

- Connect a licensed/authorized source.
- Validate payloads against `totems.schema.json`.
- Keep provenance metadata (`source`, license terms, version).

## Important

Do not import or scrape third-party 360-degree descriptions without explicit rights.
